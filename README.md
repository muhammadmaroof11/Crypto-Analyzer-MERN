# 🚀 CryptoAnalyzer - Real-Time DeFi Dashboard

**Live Demo:** [https://crypto-analyze-proj.netlify.app/](https://crypto-analyze-proj.netlify.app/)  
**Backend API:** [Hosted on Render]

## 🔥 Overview
A professional Full Stack MERN application that provides real-time cryptocurrency market data, dynamic portfolio tracking, and secure user authentication.

## 🛠 Tech Stack
- **Frontend:** React + Vite, Chart.js (Data Visualization), CSS Glassmorphism.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB (Atlas), Mongoose.
- **Authentication:** JWT (JSON Web Tokens), Bcrypt encryption.
- **API:** CoinGecko Public API.

## ✨ Key Features
- **Live Market Data:** Fetches real-time prices for Bitcoin, Ethereum, and altcoins.
- **Interactive Charts:** Dynamic performance visualization using Chart.js.
- **Secure Auth:** Complete Login/Register system with password hashing.
- **Portfolio Simulation:** Real-time valuation of assets based on live market movement.
- **Responsive UI:** Modern "Glassmorphism" design fully optimized for mobile & desktop.

## 🚀 How to Run Locally
1. Clone the repo
2. Install dependencies: `npm install`
3. Setup `.env` file (MONGO_URI, JWT_SECRET)
4. Run server: `npm run dev`
